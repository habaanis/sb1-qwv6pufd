## 🚀 Comment Utiliser les Archives

### Option 1 : Netlify Drag & Drop

1. Extraire l'archive :
   ```bash
   unzip dalil-tounes-production.zip
   # OU
   tar -xzf dalil-tounes-production.tar.gz
   ```

2. Aller sur https://app.netlify.com/drop

3. Glisser-déposer le contenu extrait (pas le dossier dist, mais son contenu)

4. Votre site sera en ligne en 30 secondes !

### Option 2 : Netlify CLI

1. Extraire l'archive

2. Déployer :
   ```bash
   cd dalil-tounes-production
   netlify deploy --dir=. --prod
   ```

### Option 3 : Serveur FTP/SFTP

1. Extraire l'archive

2. Uploader tout le contenu vers votre serveur web

3. Configurer le serveur pour servir `index.html` comme page par défaut
