====================================
IMAGE PLACEHOLDER MANQUANTE
====================================

IMPORTANT: Vous devez ajouter une image placeholder à cet emplacement:
/public/images/placeholder.jpg

Cette image sera affichée automatiquement dans les cas suivants:
- Quand le nom de fichier en base de données est vide
- Quand l'URL Supabase Storage ne peut pas être construite
- Quand une image ne charge pas correctement

RECOMMANDATIONS:
- Format: JPG, PNG ou WEBP
- Dimensions recommandées: 800x600 pixels minimum
- Poids: Moins de 200 KB pour de meilleures performances
- Contenu: Image neutre avec texte "Image non disponible" ou logo de votre application

Pour changer le chemin de l'image par défaut:
Modifiez la constante DEFAULT_IMAGE_PATH dans le fichier:
/src/lib/imageUtils.ts (ligne 11)
